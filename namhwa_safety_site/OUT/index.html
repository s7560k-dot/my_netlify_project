3. 예시

4. (중요) 건축물 바닥(슬래브) 운용 시 안전성 검토

경고 : 위험!

현재 계산기의 '지내력 데이터'는 [지반(땅)이나 기초] 에 장비를 설치할 때 사용하는 값입니다. 이 값을 건축물의
**1층 바닥(슬래브)**에 절대로 적용해서는 안 됩니다. 구조물(슬래브)의 내력은 지반의 내력과 완전히 다릅니다.

아래 내용은 1층 바닥에서 장비 운용 시 필요한 [전문적인 안전성 검토 절차]에 대한 설명이며, [반드시 자격을 갖춘
구조기술사가 수행해야 합니다.]

1단계: 정보 수집 (입력 데이터)

  • 장비 측 정보:
    • 최대 아웃트리거 반력 (R) (본 계산기의 1번 항목에서 계산 가능)
    • 아웃트리거 규격 및 정확한 배치 위치
  • 건물 측 정보:
    • 구조 도면 (구조 평면도, 상세도)
    • 구조 계산서 (설계 활하중(Live Load) 값 확인)
    • 콘크리트 설계 강도 (fck)

2단계: 전문가 검토 의뢰 (필수)

수집한 모든 자료(장비 제원, 아웃트리거 반력, 구조 도면)를 가지고 '구조기술사'에게 공식적으로 구조 안전성 검토를 의뢰해야 합니다. 현장 관리자가 임의로 판단해서는 안 됩니다.

3단계: 구조기술사의 안전성 분석

구조기술사는 다음 항목들을 계산하여 바닥의 안전성을 검토합니다.

  • 펀칭 전단 (Punching Shear) 검토 (가장 중요): 아웃트리거의 집중 하중이 슬래브를 뚫고 파괴시키지 않는지 계산합니다.
  • 슬래브 휨 (Bending Moment) 검토: 하중으로 인해 슬래브가 아래로 휘어져 균열/파괴되지 않는지 철근량을 기준으로 계산합니다.
  • 보 (Beam) / 거더 (Girder) 검토: 슬래브의 하중을 전달받는 보(Beam)가 추가 하중을 견딜 수 있는지 검토합니다.
  • 기둥 (Column) 검토: 보가 전달한 하중을 최종적으로 지지하는 기둥이 안전한지 검토합니다.

4단계: 결과 및 대책 수립

구조 검토 결과에 따라 다음과 같은 대책이 결정됩니다.

  • 안전: 바닥 구조가 충분히 튼튼하여 별도 조치 불필요 (드문 경우).
  • 하중 분산 필요: 펀칭 파괴 등을 방지하기 위해, 계산된 크기 이상의 두꺼운 강판(Steel Plate)이나 H-빔 받침대를 설치하여 하중을 넓게 분산시킵니다.
  • 동바리(Shoring) 설치 필요: 하중 분산만으로 부족할 때, 장비 설치 위치의 바로 아래층 바닥에 동바리(시스템 서포트)를 촘촘하게 설치하여 하중을 하부 구조로 직접 전달시킵니다.
  • 작업 불가: 구조적으로 보강이 불가능하거나 위험하다고 판단되는 경우.
